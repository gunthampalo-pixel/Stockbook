# Reviews

โฟลเดอร์นี้ใช้เก็บ weekly review และ quarterly review

## Naming Convention

Weekly:

```text
YYYY-WW_weekly_review.md
```

Quarterly:

```text
YYYY-QX_quarterly_review.md
```

## Purpose

- บังคับให้ทบทวน thesis ไม่ใช่แค่หาไอเดียใหม่
- ย้าย watchlist ตาม evidence
- ตรวจว่ามีหุ้น obvious/mainstream มากเกินไปไหม
- ตรวจว่า monthly trend confirm หรือเสีย
- ตรวจว่า bear case เริ่มเกิดจริงหรือไม่

## Templates

- weekly: `docs/knowledge/02_templates/weekly_review.md`
- quarterly: `docs/knowledge/02_templates/quarterly_review.md`

## Review Questions

- หุ้นไหน thesis ดีขึ้น
- หุ้นไหน thesis broken
- หุ้นไหน crowded เกิน
- หุ้นไหน chart ดีขึ้นก่อนข่าว
- หุ้นไหนต้องเขียน deep memo
