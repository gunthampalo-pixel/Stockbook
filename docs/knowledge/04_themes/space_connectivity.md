# Theme: Space Connectivity

## Core Problem

โลกต้องการ connectivity ที่ครอบคลุมพื้นที่ห่างไกล, maritime, aviation, defense, IoT และ disaster recovery มากขึ้น

## Value Chain

```text
Launch -> Satellite/Payload -> Ground Station -> Spectrum/Network -> Terminal -> Customer
```

## Bottlenecks

- launch capacity
- spectrum rights
- ground infrastructure
- terminals and antennas
- secure communication
- defense-grade reliability

## Beneficiary Types

- direct: satellite operators, launch providers, terminal makers
- indirect: component suppliers, ground station operators, defense contractors, connectivity software

## Customer Quality

สูงเมื่อเป็น government, defense/security, aviation, maritime, telecom และ enterprise infrastructure

## Key Risks

- capex intensity
- launch delays
- regulatory/spectrum risk
- constellation competition
- dilution

## What To Verify

- funded backlog
- customer contract quality
- cash runway
- unit economics
- monthly trend ไม่ใช่แค่ speculative spike
