# Theme: Cybersecurity

## Core Problem

ระบบ digital, cloud, identity, AI และ remote work ทำให้ attack surface เพิ่มขึ้น ขณะที่องค์กรต้องผ่าน compliance และลด breach risk

## Value Chain

```text
Identity -> Endpoint -> Network/Cloud -> Data Security -> Detection/Response -> Compliance
```

## Bottlenecks

- identity and access control
- cloud security posture
- data protection
- security operations automation
- compliance reporting
- AI-driven threat detection

## Beneficiary Types

- direct: platform security vendors, identity providers, cloud security providers
- indirect: compliance automation, managed security services, data governance, insurance/risk analytics

## Customer Quality

ดีมากเมื่อขายให้ enterprise, government, finance, healthcare, defense/security และ regulated industry

## Moat Signals

- integrated workflow
- high switching cost
- long-term contract
- high net retention
- mission-critical deployment

## What To Verify

- growth durable หรือเกิดจาก breach/news spike
- platform consolidation ช่วยหรือทำร้ายบริษัท
- competition จาก cloud hyperscaler
- monthly trend และ valuation
