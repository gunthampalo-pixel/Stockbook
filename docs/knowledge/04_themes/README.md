# Themes

โฟลเดอร์นี้ใช้เก็บ research theme และ value chain map

## วิธีใช้

1. เริ่มจาก pain point
2. วาด value chain
3. หา bottleneck
4. แยก direct beneficiary กับ indirect beneficiary
5. ดู customer quality
6. ค่อยหา candidate companies
7. ใช้ monthly trend filter ก่อนยกระดับเป็น Buy Candidate

## Template

ใช้ `docs/knowledge/02_templates/theme_map.md` เมื่อสร้าง theme ใหม่

## Theme Status

- `Early` - ตลาดยังไม่พูดเยอะ แต่ signal เริ่มมา
- `Developing` - demand และหุ้นบางตัวเริ่ม confirm
- `Mainstream` - คนเริ่มพูดเยอะ ต้องระวัง crowdedness
- `Euphoric` - narrative ร้อนเกิน ต้องเน้น risk
- `Broken` - thesis ไม่เดินต่อหรือ economics ไม่ดี
