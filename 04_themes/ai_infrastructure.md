# Theme: AI Infrastructure

## Core Problem

AI adoption ต้องใช้ compute, networking, power, cooling, data pipeline, security และ deployment tooling มากกว่าระบบ IT เดิม

## Value Chain

```text
Power -> Data Center -> Compute -> Networking -> Storage/Data -> Model Ops -> Enterprise Apps
```

## Bottlenecks

- power availability
- advanced chips and memory
- networking bandwidth
- cooling
- data center capacity
- model deployment and observability
- AI security and compliance

## Beneficiary Types

- direct: compute providers, cloud platforms, chip/memory/networking suppliers
- indirect: power equipment, cooling, data center REIT/operators, observability, cybersecurity, data governance
- bottleneck owners: companies controlling scarce capacity or mission-critical components

## Customer Quality

สูงถ้าลูกค้าเป็น hyperscaler, enterprise, government, regulated industry หรือ defense/security

## Crowdedness

AI เป็น theme ที่ crowded มาก ต้องหลีกเลี่ยงการเสนอแต่ obvious leaders เว้นแต่เป็น re-entry case

## What To Verify

- revenue exposure to AI จริงแค่ไหน
- margin benefit หรือ capex burden
- customer concentration
- order durability
- monthly trend ไม่ extended เกินไป
