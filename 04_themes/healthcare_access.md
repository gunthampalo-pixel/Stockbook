# Theme: Healthcare Access

## Core Problem

Healthcare มีปัญหาค่าใช้จ่ายสูง, access ยาก, workflow ซับซ้อน, labor shortage และ reimbursement friction

## Value Chain

```text
Patient -> Provider -> Diagnostics/Treatment -> Billing/Reimbursement -> Payer -> Follow-up Care
```

## Bottlenecks

- provider capacity
- diagnostics throughput
- billing and prior authorization
- clinical workflow
- remote care
- data interoperability

## Beneficiary Types

- direct: healthcare platforms, diagnostics, provider enablement software
- indirect: billing rails, workflow automation, data infrastructure, compliance tools

## Customer Quality

ต้องแยกให้ชัดว่า payer คือ patient, provider, insurer, employer, government หรือ pharma

regulated industry ทำให้ switching cost สูงได้ แต่ sales cycle และ compliance risk ก็สูงด้วย

## Key Risks

- reimbursement changes
- regulatory risk
- provider adoption ช้า
- unit economics ไม่ดี
- customer acquisition cost สูง

## What To Verify

- payer quality
- gross margin
- retention
- reimbursement dependence
- monthly trend และ liquidity
