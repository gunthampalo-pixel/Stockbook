# Memos

โฟลเดอร์นี้ใช้เก็บ memo รายหุ้น

## Naming Convention

ใช้ชื่อไฟล์แบบ:

```text
YYYY-MM-DD_TICKER_company-name.md
```

ตัวอย่าง:

```text
2026-06-04_EXAMPLE_example-company.md
```

## Recommended Template

- quick idea: ใช้ `02_templates/quick_memo.md`
- serious study: ใช้ `02_templates/deep_memo.md`
- chart/value chain summary: ใช้ `02_templates/visual_memo.md`

## Rule

ทุก memo ต้องมี:

- payer
- revenue cycle
- moat/switching cost
- financial confirmation
- monthly trend
- crowdedness
- bear case
- decision
